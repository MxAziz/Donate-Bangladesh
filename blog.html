What is Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface that represents the structure of a web page as a tree of objects. Each element (like HTML tags) on a webpage is represented as a node in this tree, which can be manipulated using programming languages such as JavaScript.

How do you select an element from the DOM ?

To select an element from the DOM, you can use methods like getElementById() for selecting by ID, querySelector() for the first matching CSS selector, and querySelectorAll() for all elements matching a CSS selector. These methods allow you to interact with and manipulate webpage elements efficiently.

What is event delegate in the context of the DOM, and why it is useful ?

Event delegation is a technique where a single event listener is added to a parent element to manage events from multiple child elements. Instead of adding separate listeners to each child, the event is captured at a higher level and handled through event bubbling.
Why is it useful?
1.Improves performance by reducing the number of event listeners.
2.Simplifies code when dealing with dynamic content.
